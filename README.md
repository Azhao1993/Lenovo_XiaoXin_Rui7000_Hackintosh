## 联想小新锐7000

-----[ 配置信息 ]-------------------------

CPU :i7-7700HQ

内存 : 8GBx2

硬盘 : 东芝XG3 256GB & 1TB 机械硬盘 5400RPM

显卡 ：NVDIA GTX1050 & Intel HD Graphics 630

声卡 ：ALC 235

有线网卡 ：RTL8168/8111

无线网卡 ：BCM94352z 联想版

-----[ 已知问题 ]-------------------------

1.FaceTime不可用

2.亮度调节最后最暗档位真的是最暗

3.HDMI插入重启

-----[ 适用系统 ]-------------------------

macOS Sierra 10.12.6

macOS High Sierra 10.13 （**将CLOVER/kexts/Other/AppleBacklightInjector.kext放入L/E或S/L/E可修复亮度**）
